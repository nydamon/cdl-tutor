# CDL State Research Agent Template

## Mission
Research CDL written test requirements for a specific US state.

## Input
- State Code: [CODE] (e.g., "NJ", "CA")
- State Name: [NAME] (e.g., "New Jersey")

## Research Tasks

### 1. Find Official CDL Handbook
- Locate official state DMV CDL handbook (PDF)
- Note URL

### 2. Test Topics
- General Knowledge
- Air Brakes
- Hazardous Materials
- Tanker Vehicles
- Passenger Transport

### 3. Extract Sample Questions (5 per topic)
- Copy actual test questions
- Note correct answers

### 4. State-Specific Rules

## Output
Save to: `data/states/[CODE].json`
Use NJ.json as reference format.
